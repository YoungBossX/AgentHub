// 功能特性

群聊式多 Agent 编程工作台

💬

群聊式任务入口

通过 @orchestrator、@frontend、@backend、@qa 把需求分发给不同角色 Agent，用户用自然语言推进任务。

🎯

计划先行的任务拆解

Orchestrator 先把需求拆成可审查计划，再分配给前端、后端、QA 等角色，避免直接黑盒改代码。

🔗

真实执行与可靠 fallback

统一适配器层支持 Codex CLI、Claude Code CLI 和 ScriptedMock。真实路径失败时，可通过明确 fallback 保持演示闭环。

📊

可审计 Diff

每个 Session 独立 worktree；Git 项目生成真实 Git Diff，非 Git target 也能基于文件快照生成变更证据。

👁️

本地预览

任务完成后生成 Vite React 预览卡片，右侧面板展示运行页面，并保留端口、URL、健康状态等证据。

🚀

诚实部署卡片

部署卡片记录本地暂存、手动交接或外部提供者阻塞状态；没有真实第三方结果时，不伪造部署成功。

// 技术栈

本地可运行的工程栈

⚡

核心组件

  • Frontend — Next.js 16 + React 19 + TypeScript 5.9
  • Backend — FastAPI 0.121 + SQLModel + SQLite
  • Styling — Tailwind CSS 4.1 + shadcn/ui 风格组件
  • Diff Viewer — Monaco Editor 0.55
🛡️

架构特性

  • Realtime — SSE + TaskRunEvent 持久化
  • Isolation — 每个 Session 独立 Git Worktree
  • Fallback — ScriptedMock 降级保证 Demo 可用
  • Guardrails — 命令白名单 + 路径保护
