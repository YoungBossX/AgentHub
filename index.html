核心功能

生产级基础架构，没有虚假演示，只有真实的代码协作和产物预览。

IM

IM 聊天式交互

左侧会话列表，支持新建/置顶/归档/搜索。单聊模式 1v1 对话，群聊模式 @多个 Agent 协作。

ORCH

主 Agent 协调器

自动理解用户意图，将复杂任务拆解并分派给合适的子 Agent，聚合产出并汇报结果。

AGT

多 Agent 接入

统一适配器层，接入 Claude Code、Codex 等主流平台，支持用户自建 Agent。

DIFF

真实 Git Diff

会话级别的 Git 工作区隔离，每一处变更都被追踪、Diff 和可验证。

PREV

产物预览与编辑

Agent 回复中内联网页预览卡片、Diff 视图卡片，点击展开全屏预览和代码编辑器。

DEP

一键部署发布

聊天中直接发送部署指令，生成预览 URL、静态站点部署或源码打包下载。
