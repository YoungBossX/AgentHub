## 3. 完成定义

- 最终演示可解释为一个本地单用户 Agent 编码工作区，并带有经过验证的编码 Agent 循环。
- 真实 Agent、兜底路径和后续路径均有当前证据或明确记录的注意事项。
- 基线文档不再与当前适配器实际情况相矛盾。
- 演示重置与恢复说明足够安全，可支持反复排练。
- 验证命令结果已记录。
- 未经用户明确请求，不得对应用代码进行任何更改、提交或推送。
