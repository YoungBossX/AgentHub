## ADDED Requirements
### Requirement: P5 工作区定位
系统 MUST 将 AgentHub 演进为本地单用户 IM 风格的多智能体编码工作区 v1，同时保留冻结的最终演示基线。

#### Scenario: P5 基线已评审
- **WHEN** 维护者审查 P5 OpenSpec 变更
- **THEN** 该变更将 AgentHub 描述为 P5 阶段的本地单用户版本
- **且** 保留了已验证的需求 -> 计划 -> 智能体执行 -> 差异对比 -> 预览 -> 模拟部署循环
- **且** 未声称具备完整的多用户即时通讯、真实部署、提供商市场、Docker 沙箱或外部即时通讯集成功能。

### Requirement: Agent 联系人及工作流模式
系统 MUST 将内置智能体暴露为 IM 风格的联系人，并 MUST 支持直接聊天和群组工作流可视化模式，而无需添加真实的多用户账户。

#### Scenario: 用户打开 P5 工作区
- **WHEN** 工作区 UI 加载完成
- **THEN** 显示内置智能体，包含头像、名称、角色、适配器类型、能力标签和状态
- **并且** 提供直接对话和群组工作流两种可视化模式
- **并且** 这些模式以本地单用户产品模式的形式呈现

### Requirement: 执行账本
系统 MUST 会为当前目标、活跃代理、最新 plan/task/run/diff/review/preview/deploy、已变更文件以及审查摘要持久化一个会话级执行账本。

#### Scenario: 代理执行后会话被重新加载
- **WHEN** 用户在某个工作流生成制品后重新加载会话
- **THEN** 该会话的执行账本可被读取
- **且** 其中引用了持久化的消息、任务、运行、制品以及变更的文件
- **且** 未暴露来自其他会话的上下文。

### Requirement: 审查 Agent 工作流
系统 MUST 在生成代码差异后创建审查制品，并且
P5 v1 审查 MUST 默认不阻塞。

#### Scenario: 编码任务产生一个 diff
- **WHEN** 一个编码 TaskRun 完成，且差异制品可用
- **THEN** 一个审查代理任务或审查工作流已运行
- **且** 审查制品以 `passed`、`warning` 或 `failed` 状态持久化
- **且** 审查制品链接到被审查的差异
- **且** 预览和模拟部署保持可用，除非后续显式策略更改非阻塞默认行为。

### Requirement: 多智能体执行追踪
系统 MUST 展示 Manager、Coding Agent、Review Agent、QA/Preview/Deploy 服务以及生成的制品，作为可见的执行追踪。

#### Scenario: 用户检查已完成的工作流
- **WHEN** 用户打开工作流追踪
- **THEN** 他们可以看到管理器规划、编码执行、差异收集、审查、预览和模拟部署步骤
- **并且** 每个步骤都链接到相关消息、任务运行或制品卡片
- **并且** 服务步骤被标记为服务而非自主代理。

### Requirement: 动态管理器规划器 V1
系统 MUST 支持针对前端变更意图的有界动态 Manager 规划器，并 MUST 保留确定性的兜底规划。

#### Scenario: 经理收到一个受支持的前端变更请求
- **WHEN** Manager 规划器处理一个受支持的前端变更意图
- **THEN** 它输出一个结构化的任务图，包含有限的任务数量、允许的角色、允许的目标文件以及依赖深度
- **并且** 同一会话内的写入任务保持串行
- **并且** 无效或不支持的规划器输出将兜底为确定性规划或安全失败。

### Requirement: 制品消息卡片 V2
系统 MUST 将 Diff、Preview、Review 和 Mock Deploy 渲染为内联消息卡片，并 MUST 支持会话范围的制品引用，以便进行后续交互。

#### Scenario: 用户引用先前的制品
- **WHEN** 用户在后续交互中选择或引用先前的制品
- **THEN** 该引用会基于当前会话进行验证
- **并且** 后续上下文标识出所选制品
- **并且** 模拟部署制品仍标记为模拟而非生产部署。

### Requirement: P5 证据规范
系统 MUST 文档仅在实际运行时记录真实的 Claude/Codex 执行情况，并 MUST 保留 `CodexAdapter`、`ClaudeCodeAdapter` 和 `ScriptedMockAdapter`。

#### Scenario: P5 端到端排练被记录
- **WHEN** P5 端到端证据已记录
- **THEN** 真实适配器执行已标注所使用的实际适配器
- **且** 兜底或脚本化行为已标注为兜底或脚本化
- **且** 当前的差异对比、预览和模拟部署流程保持不变。
